# ⚽ FIFA World Cup Dashboard (Power BI)

## 📌 Overview
This project presents an interactive Power BI dashboard analyzing FIFA World Cup historical data.

## 📊 Dashboard Preview
![Dashboard](dashboard.png)

## 🔍 Key Insights
- Goals scored by Champions vs Runner-ups
- Match trends across different years
- Growth of teams participating over time
- Distribution of winning teams

## 🛠 Tools Used
- Power BI
- Data Visualization
- Data Analysis

## 🎯 Features
- Interactive Year Filter
- KPI Cards (Goals, Matches)
- Clean and structured visuals

## 🚀 Future Improvements
- Add player-level insights
- Integrate Python for predictive analysis
